## currency-prediction-tft
## Project Overview

This project focuses on forecasting currency exchange rates using advanced time-series modeling (Temporal Fusion Transformer). The goal is to analyze financial trends and improve decision-making through data-driven insights.

## My Contribution

- Analyzed historical currency exchange data to identify trends and patterns
- Performed data preprocessing and feature engineering for financial time-series data
- Built and evaluated forecasting models to predict future currency movements
- Interpreted model outputs to understand potential market trends
- Worked with sentiment data (news) to enhance financial prediction insights

## Key Insights

- Identified patterns in currency fluctuations over time
- Evaluated prediction accuracy against actual market data
- Explored impact of external factors like news sentiment on currency trends
